# Used Car Price Prediction using Linear Regression from Scratch

## Overview

This project implements a complete Multiple Linear Regression model entirely from scratch using **NumPy**, without relying on machine learning libraries such as Scikit-learn for model training.

The objective is to predict the selling price of used cars using numerical and categorical features.

---

## Dataset

Dataset: Car Details Dataset

Features include:

- Year
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner Type

Target:

- Selling Price

---
